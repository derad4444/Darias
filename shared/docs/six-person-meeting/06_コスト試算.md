## 📊 付録: カテゴリ別相性診断コスト（2026-04-17 追加）

### 機能概要

フレンドとの BIG5 スコアを元に、4カテゴリ（友情・恋愛・仕事・信頼）それぞれについてAIがキャラクター会話を生成する機能。

```
【対象関数】diagnoseCompatibility (Cloud Function)
【モデル】gpt-4.1-mini
【呼出タイミング】ユーザーが各カテゴリの診断ボタンを押したとき（広告視聴後）
```

---

### 1カテゴリあたりのトークン数（推定）

```
【入力トークン】
- システムプロンプト（カテゴリ説明・JSON形式指示）: 約220 tokens
- BIG5口調プロフィール（双方分）: 約120 tokens
- スコアと生成指示: 約40 tokens
合計入力: 約380 tokens

【出力トークン（max_tokens: 600）】
- 会話4〜5ターン + comment + advice（JSON）: 約250 tokens（実測値）

【コスト計算（gpt-4.1-mini、日本語1文字≒1〜1.5 tokens）】
入力: 380 tokens × $0.40 / 1M = $0.000152 ≈ 0.023円
出力: 250 tokens × $1.60 / 1M = $0.000400 ≈ 0.060円
─────────────────────────────────────────
小計: 約0.083円/カテゴリ（≈¥0.08）

参考: gpt-4.1-mini料金（2026年4月時点）
入力: $0.40 / 1M tokens
出力: $1.60 / 1M tokens
```

> **6人会議との比較**: 6人会議は gpt-4o-2024-11-20 で約1.63円/回。
> 相性診断は gpt-4.1-mini で約0.083円/カテゴリ（約1/20のコスト）。
