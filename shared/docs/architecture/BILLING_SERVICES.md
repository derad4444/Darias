### 1-2. Firebase / GCP サービス

#### Firestore（データベース）

| 項目 | 内容 |
|------|------|
| 用途 | 全データの永続化（ユーザー、チャット、日記、スケジュール、メモ等） |
| 利用プラットフォーム | iOS / Web 共通 |
| 料金体系 | 読み取り/書き込み/削除の従量課金 |

| 操作 | 料金 |
|------|------|
| 読み取り | $0.06 / 10万回 |
| 書き込み | $0.18 / 10万回 |
| 削除 | $0.02 / 10万回 |
| ストレージ | $0.18 / GB / 月 |

**主なアクセスパターン:**
- キャラクター返信: 3〜5 reads/回
- 日記生成（スケジュールド）: 全ユーザー分 reads + writes/日
- 6人会議: キャッシュヒット時 2 reads、ミス時 100+ reads

#### Cloud Functions

| 項目 | 内容 |
|------|------|
| 用途 | 全バックエンドロジック実行 |
| 利用プラットフォーム | iOS / Web 共通 |
| 料金体系 | 呼び出し回数 + 実行時間 + メモリ |

| 項目 | 料金 |
|------|------|
| 呼び出し | $0.40 / 100万回 |
| 実行時間 | $0.0000025 / GB-second |
| メモリ割り当て | 512MiB〜1GiB（関数による） |

**デプロイされている関数:**

| 関数名 | トリガー | メモリ | タイムアウト |
|--------|---------|--------|------------|
| `generateCharacterReply` | onCall | 1GiB | 300秒 |
| `extractSchedule` | onCall | 512MiB | 120秒 |
| `generateBig5Analysis` | onCall | 1GiB | 300秒 |
| `generateSixPersonMeeting` | onCall | 1GiB | 300秒 |
| `scheduledDiaryGeneration` | onSchedule (毎日23:50) | 1GiB | 540秒 |
| `sendRegistrationEmail` | onDocumentCreated | - | - |
| `sendContactEmail` | onDocumentCreated | - | - |
| `validateAppStoreReceipt` | onCall | - | - |
| `validateGooglePlayReceipt` | onCall | - | - |
| `checkSubscriptionStatus` | onCall | - | - |

## 5. 月間コスト概算

### 前提条件
```
総ユーザー数: 1,000人
プレミアム課金率: 5%（50人）
平均チャット利用: 月20回/人
日記生成: 毎日（全アクティブユーザー）
```

### コスト内訳

| サービス | 月間コスト（税前） | 備考 |
|---------|-------------------|------|
| **OpenAI API** | ¥3,000〜¥15,000 | 最大コスト要因。ユーザー数・利用頻度に比例 |
| **Firestore** | ¥500〜¥3,000 | 読み書き回数に依存 |
| **Cloud Functions** | ¥100〜¥500 | 呼び出し回数 + 実行時間 |
| **Cloud Storage** | ¥50〜¥500 | 音声ファイル量に依存（現在TTS無効） |
| **Firebase Hosting** | ¥0〜¥500 | 無料枠内なら¥0 |
| **Cloud TTS** | ¥0 | 現在無効化中 |
| **Secret Manager** | **¥0**（移行済み） | .env移行により削減完了 |
| **合計** | **¥3,650〜¥19,500** | |

### 収益内訳

| 収入源 | 月間収益（税前） | 備考 |
|--------|-----------------|------|
| iOS サブスクリプション | ¥49,000 × 70% = ¥34,300 | Apple手数料30%控除後 |
| AdMob広告収益 | ¥3,000〜¥15,000 | 表示回数・CTRに依存 |
| **合計** | **¥37,300〜¥49,300** | |

### 収支（概算）

```
月間利益: ¥17,800〜¥45,650
利益率: 約70〜90%
```
