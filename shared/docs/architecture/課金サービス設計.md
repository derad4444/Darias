### 1-1. OpenAI API（最大コスト要因）

| 項目 | 内容 |
|------|------|
| 用途 | キャラクター会話・日記生成・BIG5分析・スケジュール抽出・6人会議 |
| 利用プラットフォーム | iOS / Web 共通（Cloud Functions経由） |
| 料金体系 | トークン従量課金 |

**使用モデルとティア別料金:**

| モデル | 入力料金 | 出力料金 | 用途 |
|--------|----------|----------|------|
| `gpt-4o-2024-11-20` | $2.50 / 1M tokens | $10.00 / 1M tokens | プレミアムユーザーの会話・日記生成 |
| `gpt-4o-mini` | $0.15 / 1M tokens | $0.60 / 1M tokens | 無料ユーザーの会話・日記生成・スケジュール抽出・アプリQ&A・6人会議 |

**機能別コスト（1回あたり）:**

| 機能 | ファイル | 無料ユーザー | プレミアムユーザー | 備考 |
|------|---------|------------|-----------------|------|
| キャラクター返信 | `generateCharacterReply.js` | 約¥0.023 | 約¥0.38 | 詳細は下記参照（最大値） |
| 音声生成（スピーカー押下時） | `generateVoice.js` | 非対応 | 約¥0.12（キャッシュミス時）/ ¥0（ヒット時） | TTS使用、Storage にキャッシュ |
| 日記生成（毎日23:50） | `generateDiary.js` | 約¥0.08/日 | 約¥1.21/日 | バッチ処理、最大値。ai_comment 150〜200文字に変更（2026-03-15） |
| BIG5性格分析 | `generateBig5Analysis.js` | - | 約¥8.6/ユーザー | 初回診断完了時のみ、3並列API呼び出し |
| スケジュール抽出 | `extractSchedule.js` | 約¥0.04 | 約¥0.04 | gpt-4o-mini固定、最大値 |
| アプリQ&A | `answerAppQuestion.js` | 約¥0.06 | 約¥0.06 | gpt-4o-mini固定、最大値 |
| 6人会議 | `generateSixPersonMeeting.js` | 約¥0.23（ミス時）/ ¥0（ヒット時） | 約¥0.23（ミス時）/ ¥0（ヒット時） | gpt-4o-mini固定、キャッシュあり、最大値 |
| キャラクター詳細生成 | `generateCharacterDetails.js` | - | 約¥3〜¥10/ユーザー | 2週間ごと |

**キャラクター返信のトークン内訳（1回あたり・最大値）:**

送信メッセージ構成:
```
[system] 性格特性テキスト + 性別 + 指示文 .............. ~80 tokens
[user]   履歴1のユーザー発言（max100字） ............... ~67 tokens
[assistant] 履歴1のAI返答（max100字） .................. ~67 tokens
[user]   履歴2のユーザー発言（max100字） ............... ~67 tokens
[assistant] 履歴2のAI返答（max100字） .................. ~67 tokens
[user]   現在のメッセージ（max100字）................... ~67 tokens
---
入力合計（最大値）: ~420 tokens
出力（最大値）: ~150 tokens（max_tokens: 150）
※日本語は1トークン≈1〜2文字のため、150トークンで最大250文字程度の出力が発生しうる
```

| | 無料（gpt-4o-mini） | プレミアム（gpt-4o-2024-11-20） |
|---|---|---|
| 入力 420 tokens | $0.000063 | $0.001050 |
| 出力 150 tokens | $0.000090 | $0.001500 |
| **合計** | **≒¥0.023** | **≒¥0.38** |

### 1-2. Firebase / GCP サービス

#### Firestore（データベース）

| 項目 | 内容 |
|------|------|
| 用途 | 全データの永続化（ユーザー、チャット、日記、スケジュール、メモ等） |
| 利用プラットフォーム | iOS / Web 共通 |
| 料金体系 | 読み取り/書き込み/削除の従量課金 |

| 操作 | 料金 |
|------|------|
| 読み取り | $0.06 / 10万回 |
| 書き込み | $0.18 / 10万回 |
| 削除 | $0.02 / 10万回 |
| ストレージ | $0.18 / GB / 月 |

**主なアクセスパターン:**
- キャラクター返信: 3〜5 reads/回（履歴取得・キャラ情報・BIG5スコア等）
- 日記生成（スケジュールド）: 全ユーザー分 reads + writes/日
- 6人会議: キャッシュヒット時 2 reads、ミス時 100+ reads

## 5. 月間コスト概算

### 前提条件
```
総ユーザー数: 1,000人
プレミアム課金率: 5%（50人）
平均チャット利用: 月20回/人
音声再生（プレミアム）: 月10回/人
日記生成: 毎日（全アクティブユーザー）
```

### OpenAI コスト内訳

| 機能 | 無料ユーザー（950人） | プレミアム（50人） | 月間合計 |
|------|-------------------|----------------|---------|
| キャラクター返信（月20回） | 950×20×¥0.023 = ¥437 | 50×20×¥0.38 = ¥380 | ¥817 |
| 日記生成（月30日） | 950×30×¥0.08 = ¥2,280 | 50×30×¥1.21 = ¥1,815 | ¥4,095 |
| その他（スケジュール・会議等） | - | - | ¥500〜¥2,000 |
| **OpenAI 合計** | | | **¥5,292〜¥6,792** |

### インフラコスト内訳

| サービス | 月間コスト | 備考 |
|---------|-----------|------|
| **OpenAI API** | ¥5,300〜¥6,800 | 上記参照（最大値ベース） |
